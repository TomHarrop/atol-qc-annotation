## Results

This directory contains test output from atol-qc-annotation.

It was generated by running the Snakemake workflow from
[extras](../extras/tests.smk) as follows:

```
$ snakemake --version
9.13.7

$ snakemake \
    --snakefile extras/tests.smk \
    --profile extras/profiles/default \
    test_biocontainer
```

The input data is described in [test-data](../test-data/).
